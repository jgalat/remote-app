Remote for Transmission

Control your torrents remotely.

Remote for Transmission is an open source Android app built with React Native to control your Transmission server while managing, monitoring, and fine-tuning your downloads, queue, files, peers, trackers, and speed limits.

Repository Issues Wiki Contact: remote@jg.ar

What you can do

  • Add torrents via magnet links, .torrent files, and Android share intents.
  • Inspect torrent details: files, trackers, peers, pieces, speeds, ratio, and ETA.
  • Run torrent actions: start, start now, pause, verify, reannounce, move, and remove.
  • Set per-file priorities and skip files you do not want to download.
  • Control Transmission settings: speed limits, seeding rules, queue sizes, and peer discovery.
  • Manage download directories globally and per server.
  • Use light, dark, or system theme, with optional biometric lock.
  • Receive background notifications when downloads complete.
  • Pro features include multi-server support and torrent search via Jackett or Prowlarr.
